# Lucide icons

Icons in this directory are taken from [Lucide](https://lucide.dev/),
licensed under the **ISC license**. The original SVGs have been edited
only to replace `stroke="currentColor"` with a flat stroke colour so
Slint's `Image { colorize: ... }` can recolour them per palette state.

Upstream: https://github.com/lucide-icons/lucide
